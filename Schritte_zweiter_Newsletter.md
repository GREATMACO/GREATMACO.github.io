# Zweiten Buttondown Newsletter erstellen (2 Minuten)

1. Öffne https://app.buttondown.email/ → login mit demselben Account
2. Links klick auf **"Lists"**
3. Klick auf **"Create new list"**
4. Name: `404 Collective — Survey Responses`
5. Klick auf **"Save"**
6. **Kopiere die API URL** (steht ganz oben oder unter Settings → API):
   - Sieht so aus: `https://buttondown.email/api/eps/subscribers?apikey=bd_eu1_DEIN_NEUER_KEY`

Sobald du die URL hast → schick sie mir und ich setz die letzte Variable!
