# Zweiten Buttondown Newsletter erstellen (2 Minuten)

1. Öffne https://app.buttondown.email/ → login mit deinem Account
2. Links klick auf **"Lists"** → **"Create new list"**
3. Name: `404 Collective — Survey Responses`
4. Beschreibung: `Survey-Daten für 404 Collective Pricing Research`
5. Klick auf **"Save"**
6. **Kopiere die API URL** (steht oben rechts oder unter Settings → API):
   - Sieht so aus: `https://buttondown.email/api/eps/subscribers?apikey=bd_eu1_DEIN_NEUER_KEY`

Sobald du den zweiten Newsletter erstellt und die URL hast, schick sie mir — ich richte dann in 30 Sekunden alles ein!
