# Cloudflare Global API Key erstellen (30 Sekunden)

1. Öffne https://dash.cloudflare.com/profile → "API Keys"
2. Klicke auf **"Create Key"** → **"Global API Key"**
3. Klick auf **"OK"** — der Key erscheint sofort
4. **Kopiere den Key** und schick ihn mir hier (er ist temporär)

Der Key sieht so aus: `d90c6...xxxxx`
